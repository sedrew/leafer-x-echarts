# Changelog

All notable changes to leafer-x-echarts will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.1] - 2026-04-09

### Changed
- Updated repository URLs to new GitHub location: https://github.com/sedrew/leafer-x-echarts
- Fixed package.json metadata for proper npm publishing

## [0.2.0] - 2026-04-09

### Added
- Initial release as leafer-x-echarts (renamed from leafer-echarts)
- Follows leafer-x naming conventions and standards
- Added comprehensive README with installation and usage examples
- Added Roadmap & TODO section for future features

### Features
- Render ECharts charts as SVG inside Leafer canvas
- Support for all ECharts chart types (bar, line, pie, scatter, etc.)
- Reactive updates with `updateOption()` method
- Integration with Leafer UI event system (draggable, editable)
- TypeScript support with full type definitions
- Comprehensive test suite (unit, integration, visual)

## [0.1.0] - 2026-04-05

### Added
- Initial release as leafer-echarts
- Basic ECharts rendering functionality
- SVG generation with ECharts SSR mode
- Basic test infrastructure

### Note
This version was renamed to leafer-x-echarts to follow leafer-x naming conventions.
